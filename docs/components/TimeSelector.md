# TimeSelector Component

The `TimeSelector` component is used to display a time selector. It can be used as follows:

## Usage

To use the `TimeSelector` component in your HTML file, include the following code:

```html
<div class="kai-timesel">
  <div class="kai-timesel-wrapper">
    <div class="kai-timesel-header">Select Time</div>
    <div class="kai-timesel-container">
      <input type="time" class="kai-timesel-input">
    </div>
  </div>
</div>
```

## Properties

The `TimeSelector` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `TimeSelector` component does not have any specific methods. It is a simple component that displays a time selector.
