# Button Component

The `Button` component is used to display a button. It can be used as follows:

## Usage

To use the `Button` component in your HTML file, include the following code:

```html
<button class="kai-button-input">Click me</button>
```

## Properties

The `Button` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `Button` component does not have any specific methods. It is a simple component that displays a button.
