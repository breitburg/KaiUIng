# ImageListItem Component

The `ImageListItem` component is used to display a list item with an image. It can be used as follows:

## Usage

To use the `ImageListItem` component in your HTML file, include the following code:

```html
<div class="kai-il-img">
  <div class="kai-il-img-icon">
    <img src="image.jpg" alt="Image">
  </div>
  <div class="kai-il-img-line">
    <span class="kai-il-img-primary">Primary Text</span>
    <span class="kai-il-img-secondary">Secondary Text</span>
  </div>
</div>
```

## Properties

The `ImageListItem` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `ImageListItem` component does not have any specific methods. It is a simple component that displays a list item with an image.
