# Separator Component

The `Separator` component is used to display a separator. It can be used as follows:

## Usage

To use the `Separator` component in your HTML file, include the following code:

```html
<div class="kai-separator">
  <span class="kai-separator-text">Separator Text</span>
</div>
```

## Properties

The `Separator` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `Separator` component does not have any specific methods. It is a simple component that displays a separator.
