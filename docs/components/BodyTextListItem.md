# BodyTextListItem Component

The `BodyTextListItem` component is used to display a list item with a header and body text. It can be used as follows:

## Usage

To use the `BodyTextListItem` component in your HTML file, include the following code:

```html
<div class="kai-btl">
  <span class="kai-btl-header">Header Text</span>
  <span class="kai-btl-body">Body Text</span>
</div>
```

## Properties

The `BodyTextListItem` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `BodyTextListItem` component does not have any specific methods. It is a simple component that displays a list item with a header and body text.
