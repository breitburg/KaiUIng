# SoftKey Component

The `SoftKey` component is used to display soft keys. It can be used as follows:

## Usage

To use the `SoftKey` component in your HTML file, include the following code:

```html
<div class="kai-softkey">
  <button class="kai-softkey-btn">Left</button>
  <button class="kai-softkey-btn">Center</button>
  <button class="kai-softkey-btn">Right</button>
</div>
```

## Properties

The `SoftKey` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `SoftKey` component does not have any specific methods. It is a simple component that displays soft keys.
