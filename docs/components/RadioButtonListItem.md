# RadioButtonListItem Component

The `RadioButtonListItem` component is used to display a list item with a radio button. It can be used as follows:

## Usage

To use the `RadioButtonListItem` component in your HTML file, include the following code:

```html
<div class="kai-rbl">
  <div class="kai-rbl-button">
    <input type="radio" class="kai-rbl-button-input-unfocused">
  </div>
  <div class="kai-rbl-line">
    <span class="kai-rbl-primary">Primary Text</span>
    <span class="kai-rbl-secondary">Secondary Text</span>
  </div>
</div>
```

## Properties

The `RadioButtonListItem` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `RadioButtonListItem` component does not have any specific methods. It is a simple component that displays a list item with a radio button.
