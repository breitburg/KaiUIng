# IconListItem Component

The `IconListItem` component is used to display a list item with an icon. It can be used as follows:

## Usage

To use the `IconListItem` component in your HTML file, include the following code:

```html
<div class="kai-il">
  <div class="kai-il-icon">
    <span class="kai-il-icon-unfocused"></span>
  </div>
  <div class="kai-il-line">
    <span class="kai-il-primary">Primary Text</span>
    <span class="kai-il-secondary">Secondary Text</span>
  </div>
</div>
```

## Properties

The `IconListItem` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `IconListItem` component does not have any specific methods. It is a simple component that displays a list item with an icon.
