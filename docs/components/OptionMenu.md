# OptionMenu Component

The `OptionMenu` component is used to display an option menu. It can be used as follows:

## Usage

To use the `OptionMenu` component in your HTML file, include the following code:

```html
<div class="kai-om">
  <header>Option Menu</header>
  <div class="kai-om-item">Option 1</div>
  <div class="kai-om-item">Option 2</div>
</div>
```

## Properties

The `OptionMenu` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `OptionMenu` component does not have any specific methods. It is a simple component that displays an option menu.
