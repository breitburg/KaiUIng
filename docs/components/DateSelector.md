# DateSelector Component

The `DateSelector` component is used to display a date selector. It can be used as follows:

## Usage

To use the `DateSelector` component in your HTML file, include the following code:

```html
<div class="kai-datesel">
  <div class="kai-datesel-wrapper">
    <div class="kai-datesel-header">Select Date</div>
    <div class="kai-datesel-container">
      <input type="date" class="kai-datesel-input">
    </div>
  </div>
</div>
```

## Properties

The `DateSelector` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `DateSelector` component does not have any specific methods. It is a simple component that displays a date selector.
