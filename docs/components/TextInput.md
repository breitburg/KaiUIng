# TextInput Component

The `TextInput` component is used to display a text input field. It can be used as follows:

## Usage

To use the `TextInput` component in your HTML file, include the following code:

```html
<div class="kai-text-input">
  <label class="kai-text-input-label" for="text-input">Label</label>
  <input class="kai-text-input-input" id="text-input" type="text" placeholder="Enter text">
</div>
```

## Properties

The `TextInput` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `TextInput` component does not have any specific methods. It is a simple component that displays a text input field.
