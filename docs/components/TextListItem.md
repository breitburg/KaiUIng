# TextListItem Component

The `TextListItem` component is used to display a list item with primary, secondary, and tertiary text. It can be used as follows:

## Usage

To use the `TextListItem` component in your HTML file, include the following code:

```html
<div class="kai-tl">
  <span class="kai-tl-primary">Primary Text</span>
  <span class="kai-tl-secondary">Secondary Text</span>
  <span class="kai-tl-tertiary">Tertiary Text</span>
</div>
```

## Properties

The `TextListItem` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `TextListItem` component does not have any specific methods. It is a simple component that displays a list item with primary, secondary, and tertiary text.
