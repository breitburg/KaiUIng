# IconButton Component

The `IconButton` component is used to display a button with an icon. It can be used as follows:

## Usage

To use the `IconButton` component in your HTML file, include the following code:

```html
<button class="kai-icon-button-input">
  <span class="kai-icon-button-icon-unfocused"></span>
  Button Text
</button>
```

## Properties

The `IconButton` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `IconButton` component does not have any specific methods. It is a simple component that displays a button with an icon.
