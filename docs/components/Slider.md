# Slider Component

The `Slider` component is used to display a slider. It can be used as follows:

## Usage

To use the `Slider` component in your HTML file, include the following code:

```html
<div class="kai-slider">
  <div class="kai-slider-line">
    <span class="kai-slider-header">Slider Header</span>
    <span class="kai-slider-tracker">Tracker</span>
  </div>
  <div class="kai-slider-slider-wrapper">
    <input type="range" min="0" max="100" value="50">
  </div>
</div>
```

## Properties

The `Slider` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `Slider` component does not have any specific methods. It is a simple component that displays a slider.
