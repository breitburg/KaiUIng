# Header Component

The `Header` component is used to display a header in your application. It provides a consistent look and feel for headers across your app.

## Usage

To use the `Header` component, include the following HTML in your file:

```html
<div class="kai-header">
  <h1>Header Text</h1>
</div>
```

## Properties

The `Header` component does not have any specific properties. It is a simple component that displays a header with a consistent style.

## Methods

The `Header` component does not have any specific methods. It is a simple component that displays a header with a consistent style.

## Example

Here is an example of how to use the `Header` component in your HTML file:

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="node_modules/kaiuing/dist/kaiuing.css">
  <title>Header Component Example</title>
</head>
<body>
  <div class="kai-header">
    <h1>Header Text</h1>
  </div>
</body>
</html>
```

In this example, the `Header` component is used to display a header with the text "Header Text". The CSS file from KaiUIng is included to apply the necessary styles.
