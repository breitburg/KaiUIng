# Dialog Component

The `Dialog` component is used to display a dialog. It can be used as follows:

## Usage

To use the `Dialog` component in your HTML file, include the following code:

```html
<div class="kai-dialog">
  <div class="kai-dialog-wrapper">
    <div class="kai-dialog-header">Dialog Title</div>
    <div class="kai-dialog-container">
      <p>Dialog content goes here.</p>
      <button class="kai-button-input">OK</button>
    </div>
  </div>
</div>
```

## Properties

The `Dialog` component does not have any specific properties. It relies on the CSS classes to style the component.

## Methods

The `Dialog` component does not have any specific methods. It is a simple component that displays a dialog.
